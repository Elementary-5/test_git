#自动编译工具

可以完成code samples的自动下载并记录编译结果

##程序结构

分如下几个模块

1. 爬虫
2. 下载
3. 编译
4. 截图
5. 其他
###模块简介
1. 爬虫
 
	可根据输入的关键字，去选择需要下载的项目。

2. 下载
	
	根据爬虫的结果选择相应的项目下载

3. 编译
	
	根据爬虫的结果编译相应的项目并生成log xml文件记录相应的结果

4. 截图
	
	对能运行的程序进行截图

5.其他
	
	UseDemo 里面演示了可以怎么样使用这个工具
	xmllogger 为了生成log xml文件才写的一个dll
	xmlreader 用来读取xml文件里面的内容，便于查看
